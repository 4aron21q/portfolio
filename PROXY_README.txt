Простой многопоточный TCP-прокси на Python

Учебный скрипт для перенаправления сетевого трафика. Принимает входящие соединения на локальном порту и прозрачно пересылает данные на указанный удаленный сервер в обе стороны.

Как запустить:
1. Запусти скрипт: python proxy.py
2. Введи IP-адрес целевого сервера (например, 127.0.0.1)
3. Введи порт целевого сервера (например, 8000)
4. Подключи клиентскую программу или браузер к адресу 127.0.0.1:8080

Как это работает:
- Скрипт слушает порт 8080 и принимает новые подключения.
- Для каждого клиента создается соединение с целевым сервером.
- Запускаются два независимых потока: один пересылает данные от клиента к серверу, второй — от сервера к клиенту.
- При разрыве соединения потоки завершаются, сокеты корректно закрываются.

Особенности:
- Написан на стандартных библиотеках Python (socket, threading)
- Поддерживает двунаправленную передачу данных без блокировок
- Обрабатывает отключение клиентов без падения основного процесса
- Использует daemon-потоки для автоматической очистки ресурсов

Важно:
- Код создан исключительно в образовательных целях
- Не предназначен для использования в продакшене или обхода средств защиты
- Используй только на собственных системах или с явного разрешения владельца сети
