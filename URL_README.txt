Простой сканер веб-уязвимостей на Python

Учебный скрипт для автоматической проверки веб-приложений на базовые уязвимости. Написан для изучения основ AppSec, работы с HTTP-запросами и автоматизации ручного тестирования.

Установка:
pip install requests

Как запустить:
1. Запусти скрипт: python scanner.py
2. Введи целевой URL (обязательно с http:// или https://)
3. Дождись завершения проверки и изучи вывод в консоли.

Что проверяет скрипт:
- Доступность чувствительных файлов (.env, .git/config, robots.txt)
- Reflected XSS через параметр ?q= (поиск отражения пейлоада в ответе сервера)
- Error-based SQLi через параметр ?id= (поиск типичных сообщений об ошибках баз данных)

Особенности реализации:
- Использует библиотеку requests для отправки HTTP-запросов
- Добавлены задержки между запросами для снижения нагрузки на цель
- Реализована базовая обработка сетевых ошибок и таймаутов
- Результаты выводятся в консоль в виде краткого отчета

Важно:
- Скрипт создан исключительно в образовательных целях
- Используй его только на своих проектах или ресурсах с разрешения владельца
- Результаты проверки носят предварительный характер и требуют ручной верификации
