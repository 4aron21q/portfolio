Многопоточный чат-сервер на C

Учебный сервер для обмена сообщениями между несколькими клиентами. Написан на чистом C с использованием POSIX сокетов и библиотеки pthread.

Компиляция:
gcc chat_server.c -o chat_server -lpthread

Запуск:
./chat_server

Как подключиться:
Открой несколько терминалов и подключись к серверу с помощью netcat:
nc 127.0.0.1 9000
Вводи сообщения в любом терминале. Они мгновенно появятся у всех остальных подключенных пользователей.

Как это работает:
- Сервер слушает порт 9000 и принимает входящие TCP-соединения.
- Для каждого клиента создается отдельный поток обработки.
- Сообщения от клиента читаются в цикле и через функцию broadcast рассылаются всем остальным участникам чата.
- Доступ к общему списку клиентов защищен мьютексом (pthread_mutex_t), что предотвращает конфликты при одновременном подключении или отправке данных.
- При отключении клиента поток завершается, сокет корректно закрывается.

Особенности реализации:
- Полностью на стандартных библиотеках Linux/POSIX
- Потокобезопасная работа с общим массивом подключений
- Ограничение на количество одновременных клиентов (10)
- Автоматическая очистка ресурсов потоков через pthread_detach

Важно:
- Код написан в учебных целях для изучения многопоточности и работы с сетевыми сокетами в C.
- Не предназначен для промышленного использования или продакшн-среды.
- Для компиляции и запуска требуется Linux, macOS или WSL с установленным GCC.
