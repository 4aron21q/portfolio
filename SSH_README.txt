Простой интерактивный SSH-клиент на Python
Учебный скрипт для подключения к SSH-серверу и работы в командной строке удаленной машины. Написан с использованием библиотеки paramiko и многопоточности.
Установка:
pip install paramiko
Как запустить:
Открой скрипт и измени данные в последней строчке:
ssh_client("IP_СЕРВЕРА", 22, "Логин", "Пароль", "")
Запусти скрипт:
python ssh_client.py
Вводи команды в консоль. Для выхода напиши exit.
Как это работает:
Скрипт подключается к SSH и открывает интерактивный shell.
Запускает фоновый поток, который постоянно читает вывод от сервера и сразу печатает его в консоль.
Основной поток ждет ввод пользователя, отправляет команды на сервер и добавляет перевод строки.
При вводе exit соединение корректно закрывается.
Важно:
Код создан только в образовательных целях.
Используй его только на своих серверах или с разрешения владельца.
В реальном проекте лучше использовать готовые CLI-инструменты или проверенные библиотеки для работы с SSH.
